# Degree 01 — Swift Overview

**Status:** In progress (Phase 0 complete, Phase 1+ pending)
**Slug:** `01-swift-overview`
**Audience:** LLM agents (autonomous coding agents)
**Prerequisites:** none (this is the first Swift degree in the ecosystem)
**Estimated POCs:** 6 (L1–L5 + capstone)

## Goal

Teach an LLM agent how to build real server-side and CLI Swift software end-to-end: from `swift package init` to a deployable, tested, agent-callable HTTP service.

## Deliverables

- Six built, tested, committed POCs (`03-pocs/L1-*` through `L-capstone-*`).
- Research corpus (`01-research/`) covering language, SwiftPM, swift-testing, swift-argument-parser, Swift concurrency, Hummingbird.
- Planning artifacts (`02-planning/`).
- Distilled knowledge (`05-distillation/`) — gotchas, patterns, anti-patterns, playbooks, before-you-build, decision-records — every claim with evidence.
- Skill pack (`06-skill-pack/`) — navigable, self-contained.
- Readiness assessment (`07-evaluation/readiness.md`).
